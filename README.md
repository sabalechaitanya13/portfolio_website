# 🌐 Personal Portfolio Website

A modern, responsive, and animated personal portfolio website built using pure HTML, CSS, and JavaScript.  
Designed with a premium dark UI, glassmorphism effects, animated sections, interactive particles, and smooth transitions to create a professional recruiter-focused portfolio experience.

## ✨ Features

- Modern premium UI/UX
- Fully responsive design
- Animated hero section
- Interactive particle background
- Smooth scrolling navigation
- Typing text animation
- Custom cursor effects
- Glassmorphism cards
- Animated timeline experience section
- Skills showcase with app-style icons
- Projects section
- Certifications section
- Contact section
- SEO optimized meta tags
- Lightweight & fast loading
